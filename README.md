# Isometric Weather Game

AOE(Age of Empires) 게임 방식의 시야(Fog of War) 시스템과 비오는 날씨 효과가 있는 아이소메트릭 게임입니다.

## 🎮 게임 특징

### 1. **Fog of War 시스템** 🌫️
- AOE 게임처럼 유닛의 시야 범위 내에서만 지형과 적군이 보입니다
- **검은 안개**: 아직 탐험하지 않은 영역
- **어두운 안개**: 탐험했지만 현재 보이지 않는 영역
- **밝은 영역**: 현재 보이는 영역

### 2. **날씨 시스템** 🌧️
- 시간에 따라 날씨가 변합니다
- **맑음**: 기본 상태
- **비**: 중간 강도의 비
- **폭우**: 강한 비
- 비의 강도에 따라:
  - 파티클 효과 조절
  - 조명 강도 변경
  - 사운드 볼륨 조절

### 3. **아이소메트릭 카메라** 📷
- 45도 각도의 아이소메트릭 뷰
- 플레이어를 부드럽게 따라다니는 카메라

### 4. **유닛 조작** 🚶
- 좌클릭: 유닛 선택
- 우클릭: 선택한 유닛을 클릭한 위치로 이동
- 유닛은 자동으로 이동 방향을 바라봅니다

## 🛠️ 주요 클래스

### FogOfWarSystem
- 맵을 그리드로 나누어 시야 관리
- 각 유닛의 시야 범위를 원형으로 계산
- 탐험한 영역과 현재 보이는 영역 구분

### WeatherSystem
- 시간 기반 날씨 사이클
- 파티클, 조명, 오디오 동적 조절
- 3가지 날씨 상태 관리

### Unit
- 유닛 이동 및 로테이션
- Fog of War와 연동하여 시각성 변경
- 애니메이션 제어

### IsometricCamera
- 아이소메트릭 각도 계산
- 타겟 추적
- 부드러운 카메라 이동

## 🎮 조작법

| 입력 | 동작 |
|------|------|
| 좌클릭 | 유닛 선택 |
| 우클릭 | 선택 유닛 이동 |

## 🚀 시작하기

1. Unity 프로젝트를 생성합니다
2. 이 저장소의 파일들을 Assets 폴더에 복사합니다
3. Scene을 생성하고 GameManager를 배치합니다
4. 필요한 프리팹들을 만들고 설정합니다
5. Play 버튼을 누르고 게임을 시작합니다
